> Typing Animation

Used to create a dynamic and immersive text animation in our landing page.

    
    const typed = new Typed(".auto-type", {
        strings: ["Prototyping", "Coding", "Debugging"],
        typeSpeed: 140,
        backSpeed: 30,
        backDelay: 2000,
        cursorChar: "_",
        loop: true
    })

It place a strings on the selected HTML element class .auto-type that automatically loops between inputed strings in the JavaScript code. Selectors varies between HTML Element, Class, or Id.

You can see Typed.JS documentation for more details.



> SVG/Image Animation

An SVG or Image animation that lowkey kinda nice

    
    const svgLeft = anime({
        targets: '#svg-left',
        translateX: [0, 180, 0],
        duration: 2000,
        autoplay: true,
        easing: 'easeInOutElastic',
        delay: 600,
        loop: true
    })

    const svgRight = anime({
        targets: '#svg-right',
        translateX: [0, -180, 0],
        duration: 2000,
        autoplay: true,
        easing: 'easeInOutElastic',
        delay: 600,
        loop: true
    })

Works almost the same as earlier, placed a JS code in the selected HTML element (in this case SVG/IMG) then put an animation to it.

As usual, you can learn more by visiting Anime.JS documentation.



> Stagger Animation

What's a stagger? basically a multi-animated child element in the same parent. You can see the examples of our stagger animations here.

    
    const labelAnimation = anime({
        targets: '.badge .labelStagger',
        translateY: [0, 20, 0],
        easing: 'easeInOutCubic',
        autoplay: true,
        loop: true,
        duration: 2000,
        delay: anime.stagger(100)
    })

.badge is a parent class, .labelStagger are the children.
